# Factor Backtest Framework

A Python-based quantitative research framework for cross-sectional equity factor investing.

## Features

- Historical market data collection using Yahoo Finance
- Factor construction and normalization
- Long-short portfolio formation
- Performance evaluation
- Information Coefficient (IC) analysis

## Factors Implemented

- Momentum Factor
- Reversal Factor
- Volatility Factor

## Backtest Methodology

Stocks are ranked cross-sectionally each trading day.

- Long top 20%
- Short bottom 20%
- Equal-weighted portfolio
- Daily rebalancing

## Performance Metrics

- Annual Return
- Annual Volatility
- Sharpe Ratio
- Maximum Drawdown
- Information Coefficient (IC)

## Technologies

- Python
- Pandas
- NumPy
- Yahoo Finance
